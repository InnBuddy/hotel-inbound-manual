# OTAコンサル 営業リスト 作成ガイド

## 対象: 日本全国の客室数5〜50室の宿泊施設

---

## 方法A: 楽天トラベルAPIで自作する（無料・推奨）

### 手順

**1. APIキーを取得**
- 楽天デベロッパー https://developers.rakuten.com/ に登録
- 「アプリ作成」→ 「新規アプリ」→ アプリIDを発行

**2. Pythonスクリプトを実行**

```bash
# 依存パッケージインストール
pip install requests

# 環境変数にAPIキーを設定して実行
export RAKUTEN_APP_ID='ここに取得したアプリID'
python rakuten_hotel_scraper.py
```

**3. 出力**
- `ota_sales_list_rakuten.csv` が生成される
- 列: 施設名, 都道府県, 住所, 電話番号, WEBサイト, 客室数, 施設タイプ

### メリット・デメリット
- 無料で使える（API利用回数に制限あり）
- 施設名・電話番号・Webサイト・住所が取得できる
- 客室数の情報は楽天APIの仕様上、全施設で取得できない場合がある
- 1都道府県ずつ取得するため完了までに時間がかかる

## 方法B: フレリス(frelis.net)の完成データを購入する（有料・確実）

### 商品: 「国内宿泊施設・ホテル（全国）」
- URL: https://frelis.net/data/40322/
- 価格: **41,800円（税込）**
- 件数: **41,724件**
- データ: 施設名, 都道府県, 郵便番号, 所在地, 電話番号, FAX, **総部屋数**
- サンプル: 100件の無料サンプルあり（マスク表示だが形式確認可）

### 該当件数
| 客室数 | 件数 |
|--------|------|
| 1〜9室 | 21,338件 |
| **10〜49室** | **12,175件** ← 貴社ターゲット |
| 50〜99室 | 3,274件 |
| 100〜499室 | 4,785件 |

購入後、客室数「10〜49室」でフィルタすれば**約12,000件**の対象リストが即座に得られます。

## 方法C: BODIK政府オープンデータ（無料）

各自治体が公開する「旅館業許可施設一覧」のCSV。
- URL: https://odm.bodik.jp/tl/dataset/272159_ryokan
- 全国版は1ファイルにまとまっておらず、自治体ごとに個別取得が必要
- 客室数の情報は含まれない

→ 網羅性が低いため、**補助的な利用**が現実的です。

---

## おすすめの進め方

1. **まず楽天APIスクリプト**を実行してデータを取得（無料）
2. 足りない部分を**フレリス**の購入データで補完
3. 両方を統合して1つの営業リストにまとめる

合計で約15,000〜20,000件のOTAコンサル営業先リストが完成します。
